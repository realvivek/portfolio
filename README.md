# portfolio

Personal portfolio / informal resume of Vivek Yalamanchili — Principal Engineer,
private 5G · edge AI · smart city infrastructure.

Static site (HTML/CSS/JS, no build step). Deployed on Render; pushes to `main`
go live automatically.

## Structure

```
index.html          entry point
assets/style.css    all styles
assets/app.js       projects data, spectrum figure, page behavior
```

## Adding a project

Append an entry to the `PROJECTS` array in `assets/app.js`.
